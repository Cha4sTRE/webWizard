Sobre nosotros...

Somos una pequeña empresa de creación y diseño de paginas web que ofrece sus servicios a pequeños emprendedores que apenas salen a luz, pero que aun no saben hacerse un lugar dentro de esta nueva era digital

Objetivo....

Nuestro objetivo es ofrecer oportunidades, aportando nuestros conocimientos, de impulsar los proyectos de los clientes para que logren alcanzar la modernización o ser parte del mercado competitivo.

Ayudar a crear una imagen o apariencia, dar una buena experiencia de usuario para los clientes del negocio sobre el que brindaremos nuestro apoyo es el objetivo, crear confianza y seguridad

nuestro servicio incluye:

- planificacion de diseño
- desarrollo atravez de recursos y herramientas nativas
- diseño de base de datos
- implementacion de mecanismos de seguridad
- hosting y dominio (paga)

tecnologias disponibles:

- node js
- react js
- angular js
- mongoDB
- express
